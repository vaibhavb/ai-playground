# ai-playground

**AI By Hand:** Interactive JavaScript learning app for neural network concepts. Input exercises in Markdown, solve problems on-screen, and get automatic validation—adaptable to any NN concept from ReLU to backpropagation with creative, distinctive UI design.
[Launch the experiment](ai-by-hand/) to open the interactive learning tool.

**Stock Hedge Simulator:** React + TypeScript hedge lab with "Financial Terminal Noir" UI, featuring Playfair Display × JetBrains Mono typography, dark brutalist aesthetics, live Yahoo quotes, OptionsProfitCalculator put-chain autofill, and tax-aware analytics for NVDA, BRK.B, ASML, MSFT, or custom tickers. Rebuilt with kinetic data visualization, atmospheric effects, and Bloomberg-inspired design.
[Launch the experiment](stock-hedge-simulator/) to open the interactive simulator and read its build notes.
